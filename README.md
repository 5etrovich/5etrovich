### 📁 Проекты

#### 🔹 [bwt-rle-compress](https://github.com/5etrovich/bwt-rle-compress)
[![CI](https://github.com/5etrovich/bwt-rle-compress/actions/workflows/ci.yml/badge.svg)](https://github.com/5etrovich/bwt-rle-compress/actions/workflows/ci.yml)

Утилита сжатия файлов на C++20: BWT (преобразование Барроуза–Уилера) + RLE. Блочная обработка, big-endian заголовки, корректный roundtrip на любых бинарных данных.

#### 🔹 [custom-vector-cpp](https://github.com/5etrovich/custom-vector-cpp)
[![CI](https://github.com/5etrovich/custom-vector-cpp/actions/workflows/ci.yml/badge.svg?branch=master)](https://github.com/5etrovich/custom-vector-cpp/actions/workflows/ci.yml)

Шаблонный контейнер `Vector<T>` с итераторами произвольного доступа и move-семантикой.  
25 unit-тестов, CI с ASan/UBSan, покрытие кода, cppcheck.

#### 🔹 [rsa-cpp](https://github.com/5etrovich/rsa-cpp)
[![CI](https://github.com/5etrovich/rsa-cpp/actions/workflows/ci.yml/badge.svg?branch=master)](https://github.com/5etrovich/rsa-cpp/actions/workflows/ci.yml)

Реализация RSA на чистом C++98 без внешних библиотек (кроме OpenSSL для BigNum). BigNum, генерация простых чисел, модульная арифметика.

#### 🔹 [elgamal-cpp](https://github.com/5etrovich/elgamal-cpp)
[![CI](https://github.com/5etrovich/elgamal-cpp/actions/workflows/ci.yml/badge.svg?branch=master)](https://github.com/5etrovich/elgamal-cpp/actions/workflows/ci.yml)

Криптосистема Эль-Гамаля на C++98. Дискретное логарифмирование, криптография с открытым ключом.

Android-приложение (Kotlin) для учёта коммунальных платежей. Многозонные тарифы, локальное хранение (JSON), Material Design. Используется реальными людьми.
